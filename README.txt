By Rodrigo García-López
Created Sept 29th, 2017
This repository contains scripts that can be used for constructing a viral database from scratch using the Viral fraction of the INSDC data as uploaded in the EBI.
The methods are auxiliary for a book chapter on the subject by the author published by Elsevier.
The scripts are the author's propiety and can be distributed under the creative commons license.
